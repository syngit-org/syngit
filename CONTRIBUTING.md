# Contributing guidelines

## Quick start

```sh
## List commands
make help

## 1. Run the controller (auto-create kind cluster)
make run

## 2. Cleanup resources
make run-reset

## 3. Run the tests
make test
```

Please refer to [this](https://github.com/syngit-org/syngit/wiki/%F0%9F%A4%9D-Contribute) wiki page.
